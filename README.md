# Nexus
Welcome!

In this repository we intend to publish Nexus SDV, a reference implementation for a connected vehicle platform for the Software Defined Vehicle.

## Getting started
Nexus is intended to be deployed into a project within your Google Cloud tenant. Deployment is automated leveraging Terraform.

## Vision
Providing a connected vehicle platform reference implementation, designed for leveraging (Gen)AI at scale, focusing on the developer needs to iterate fast.
